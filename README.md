# Hands-On Machine Learning Practice

Personal practice repository following **"Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow"** by Aurélien Géron.

## About

This repo contains my code, notes, and exercises as I work through the book's concepts including:

- Supervised Learning (Regression, Classification)
- Unsupervised Learning (Clustering, Dimensionality Reduction)
- Neural Networks & Deep Learning
- Scikit-Learn pipelines
- Keras & TensorFlow fundamentals
